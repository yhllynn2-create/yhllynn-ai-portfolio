### 销售 AI 提效 · 企微运营话题引擎 + 每日拓客引擎（已落地 Skill 合集）

> 面向 B2B 销售的「AI 提效」工作流合集，把一线最费时的两件事——**维系客户**与**找新客户**——做成可复制、可自动化的引擎。两个子项目均以 WorkBuddy Skill 形式落地，采用「1 人跑引擎、全员复制分发」的中心化架构，绕开每人部署。

**子项目一 · 销售企微运营话题引擎**

- 每周自动联网抓取最新素材，生成 5 张可直接复制发企业微信的「话题 + 话术」卡片，帮销售暖场、建立联系、挖掘商机
- 内置 8 类话题矩阵、五段式话术公式、四条内容红线、制造业弹药库与种子范例
- 入口：[sales-ai-boost/销售企微运营/README.md](./sales-ai-boost/销售企微运营/README.md) ｜ 实样与方案：[sales-ai-boost/销售企微运营/示例/](./sales-ai-boost/销售企微运营/示例/)

**子项目二 · 每日拓客引擎**

- 一条命令产出一批"大而近期有信号"、且**绝不撞上在库存量客户**的新客户，附标准化《公司调研报告》+ 汇总表
- 护城河是"强排除校验"：对候选与在库名单做归一化 + 双向子串 + 模糊兜底 + 别名穿透四重比对，零命中才过
- 入口：[sales-ai-boost/每日拓客/README.md](./sales-ai-boost/每日拓客/README.md) ｜ 机密客户名单一律不入库

**核心亮点速览（合集）**

- 8 类话题矩阵 + 五段式话术公式，覆盖联想好消息 / 行业趋势 / 政策补贴 / 节气关怀 / AI 算力
- 强排除四重校验，保证"新客"不撞存量；方法开源、客户名单本地保管
- 已每周自动化运行；1 人生产、全员零部署分发
