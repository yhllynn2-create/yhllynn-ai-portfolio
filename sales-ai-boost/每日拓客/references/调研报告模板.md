# 调研报告模板（7 节）

> 每家入选公司按此 7 节结构撰写。以下以虚构公司"**示例智能装备集团股份有限公司**"演示占位写法；实际使用时替换为真实调研数据，并标注来源。

---

### 示例智能装备集团股份有限公司

**一、公司概况**
示例智能装备集团股份有限公司，注册地 XX 市 XX 区，主营工业自动化装备/智能产线；品牌"示例装备"。国家级高新技术企业，是国内 XX 领域头部企业之一。（一句话说清：做什么、在哪、行业地位）
> **采集提示（喂打分卡）**：补一句**成立时间/经营年限**与**当前人员规模**，这是"经营稳定性"与"人员规模"两维度的直接依据。

**二、工商与股权**
法定代表人/董事长 XXX；注册资本约 X 亿元；企业类型股份有限公司；控股股东/实控人（如有公开信息）；是否上市（上市板块/代码）。（股权结构、是否有国资/产业资本背书）
> **采集提示（喂打分卡）**：补**工厂/产能分布**（总部基地 + 外地基地数量、是否自有产能、是否在扩产），这是"工厂/产能分布"维度的依据——纯贸易/轻资产公司与多基地扩产公司价值差异极大。

**三、财务表现**
近 1–3 年营收/净利润趋势（若有公开财报或招股书）；毛利率水平；当前所处阶段（规模化扩张 / 扭亏为盈 / 稳定增长）。（无公开财务时可写"未披露，据行业估算"并标注）

**四、招聘信号**
近 30 天在招岗位与人数（研发/工艺/质量/销售/交付等）；招聘扩张指向的业务动作（扩产/新线/出海）。（招聘激增 = 新人 → 新设备/新终端需求信号）

**五、融资/上市/股价信号**
近期融资轮次与金额、投资方；IPO 受理/递表/上市进度；若已上市，近期股价/市值异动与原因。（有资金 = 有采购预算信号）
> **采集提示（喂打分卡）**：末尾补一句**未来赛道趋势判断**（所在行业是朝阳高景气 / 成熟稳定 / 下行），这是"未来赛道趋势"维度的依据。

**六、IT 设备与算力需求**
> 本节按"业务动作 → 设备需求"推导（通用方法论），把具体硬件替换为**你公司产品线**即可。

结合其业务，推导对终端/工作站/服务器/算力的潜在需求：
- 研发设计：图形工作站、高性能移动终端
- 产线/办公：商用台式机、轻量服务器
- AI/仿真：GPU 工作站、边缘算力
（按"业务动作 → 设备需求"逻辑推导，避免空泛；示例以某 B2B 厂商商用 PC/工作站为例，落地时按自家产品改写）

**七、针对你公司产品线的切入建议**
> 本节**参数化**：把"示例厂商"替换为"你所在公司的产品线"。以下是示例写法，请按自家产品改写。

推荐组合（示例：以某 B2B 厂商商用 PC/工作站为例）：
- 研发/算法人员：高端移动工作站 + 商用笔记本
- 产线办公：商用台式机
- 研发数据与 PLM：轻量机架服务器
- 切入窗口：以"近期融资扩产 + 招聘激增"为钩子，对接其 IT/行政/产线负责人
- 话术锚点：批量采购折扣 / TCO 更省 / 统一运维

## 写作规范

- **公司名**：只用工商注册全称，不附加股票代码（如 `(002869.SZ)`）或品牌别名（如 `（DemoBrand）`）。注册名本身含地名括号的保留（如 `示例创新技术（深圳）有限公司`）。
- **来源可溯**：每条关键结论（融资额、营收、招聘数、注册地）尽量附来源链接/平台，宁缺毋滥。
- **时效优先**：只写近期（近 30–90 天）有效信号；过期信号不写或标注"历史"。
- **不编造**：查不到的财务/股权数据写"未公开披露"，不臆测。
- **脱敏**：对外版本不出现任何真实存量客户名；示例公司均为虚构。

## 与"商业价值打分卡"的衔接

本 7 节报告的每一个事实，都对应 `商业价值评估维度与打分卡.md` 的一个维度。写好 7 节后，把事实转成 1–5 分即可跑出价值层级：

| 报告节 | 对应打分维度 |
|--------|--------------|
| 一（含成立时间/人员） | 经营稳定性、人员规模 |
| 二（含工厂产能） | 工厂/产能分布、资本化/上市 |
| 三 | 盈利情况 |
| 四 | 近期招聘 |
| 五（含未来趋势） | 资本化/上市、近期发展动能、未来赛道趋势 |
| 一（行业地位） | 行业地位 |
| 六 | 主营业务与卖方匹配度（设备/算力密集度） |

- **拓客模式**：打分卡用于本批线索分层（高价值优先打）。
- **研判模式**：打分卡是核心产出，写法见 `价值研判报告模板.md`（在 7 节后追加 ⑧ 打分卡 + ⑨ 结论）。
