# 信号源清单（候选采集）

> 围绕"**有资金 / 有扩编 = 有采购**"逻辑，从以下信号源捞候选。优先选近期（近 7–30 天）出现信号的"大公司"。

## 一、招聘信号（新人 → 新设备/新终端）
| 信号 | 去哪看 | 含义 |
|------|--------|------|
| 大量招人（研发/工艺/质量/销售） | BOSS 直聘、猎聘、前程无忧 | 业务扩张，需配 PC/办公设备 |
| 新建基地/产线招聘 | 招聘网站 + 新闻 | 新场地 → 整批设备采购 |
| 校招大规模启动 | 各校就业网、牛客 | 新人批量入职 → 笔记本采购 |

搜索示例：`"XX行业" 招聘 扩产 2026`、`"城市" 制造业 急招 研发`

## 二、融资 / IPO 信号（有资金 → 有预算）
| 信号 | 去哪看 | 含义 |
|------|--------|------|
| 一级市场新融资（天使~Pre-IPO） | IT 桔子、36 氪、投资界、企名片 | 弹药充足，采购窗口打开 |
| IPO 受理 / 递表 / 上市 | 证监会/交易所官网、东方财富、同花顺 | 募资扩产，大单前兆 |
| 战略入股（产业资本） | 公司公告、财经媒体 | 绑定供应链，采购确定性高 |

搜索示例：`"细分赛道" 融资 2026`、`"城市" 企业 IPO 受理`

## 三、招投标 / 重大合同 / 扩产新闻（订单 → 产能 → 设备）
- 政府招投标平台、采招网：中标/大单
- 地方发改委/工信局公示：新项目备案、扩产用地
- 公司官微/官网新闻：重大合同、新基地动工

## 四、政策 / 行业红利（补贴 → 换新）
- 设备更新 / 以旧换新补贴、智能制造专项、专精特新补贴
- 搜索：`"城市" 设备更新 补贴 2026`、`制造业 数字化 转型 政策`

## 五、交叉验证源（第 4 步深核验用）
- **工商/股权/融资**：企查查 MCP、天眼查 MCP（若已连通）、WebSearch
- **注册地核实**：企查查/天眼查工商信息、公司官网"联系我们/关于我们"
- **财务**：招股书（交易所官网）、年报（上市公司）、企查查财务栏

## 六、筛选漏斗（建议）
```
信号源捞候选（宽）
   ↓ 规模门槛过滤（上市/千人级/独角兽，剔除小公司）
   ↓ 强排除校验（vs 在库名单，零命中）
   ↓ 多源深核验（注册地/规模/信号真实）
   ↓ 去重（本批 + 历史已拓客）
   ↓ 入选，写 7 节报告
```

> 原则：**在精不在多**。每家做深度调研，不堆量；小公司、信号过期、撞库公司一律不纳入。
